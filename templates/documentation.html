User Documentation

When you click on the "Scan Available WiFi" button, it initiates a scan of all available WiFi signals. The results are then listed, providing details such as SSID name, signal strength, frequency, and encryption key for each WiFi network. Importantly, the scan filters out any blocked WiFi signals during the process.

Clicking on the "Available WiFi Bar Graph" link will plot a graph of all available WiFi networks, showcasing their signal strengths. This visual representation helps you quickly assess the WiFi landscape and identify networks with stronger signals.

Clicking on the "Block" link allows you to block a specific WiFi network. Enter the SSID of the WiFi you want to block, and the system will block it. If the entered WiFi signal is not found, you'll receive a message indicating that the WiFi signal was not found.

Clicking on the "Unblock" link allows you to unblock a WiFi network that was previously blocked. Enter the SSID of the WiFi you want to unblock, and the system will restore its availability. If the entered WiFi signal is not found, you'll receive a message indicating that the WiFi signal was not found.

Clicking on the "More Info" link allows you to enter the SSID of a WiFi network and view detailed information corresponding to that network. This includes additional details such as signal strength, frequency, encryption key, and any other relevant information available for the specified WiFi network.

Clicking on the "Blocked WiFi List" link displays a list of all blocked WiFi networks. If there are no blocked WiFi networks, a message will indicate that the blocked WiFi list is empty, and there is nothing to display.

Clicking on the "Remove All Blocked WiFi" link removes all WiFi networks from the blocked list. A confirmation message will be displayed, and the blocked WiFi list will be empty if successful.

Clicking on the "Connect to a WiFi" link opens a new page where you can enter the SSID and password of the WiFi network you want to connect to. If the provided credentials are correct, the system will connect to the WiFi. In case the entered WiFi is not found, a message will be displayed indicating that the entered WiFi was not found.

Clicking on the "Ping to Host" link opens a new page where you can enter the IP address or domain of the host you want to ping. If the entered information is correct, a graph displaying ping number and time will be shown. Additionally, details such as host name, minimum, maximum, and average time along with packet loss will be listed.

Clicking on the "Internet Speed Test" link initiates a test of the current network's internet speed. After the test is complete, a link will be provided containing the results. Users can open the link to view details such as download speed, upload speed, and other relevant information about the internet speed of the connected network.
