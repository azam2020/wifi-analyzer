User Documentation

Clicking on the "Connect to WiFi" link opens a new page where you can enter the SSID and password of the WiFi network you want to connect to. If the provided credentials are correct, the system will connect to the WiFi. In case the entered WiFi is not found, a message will be displayed indicating that the entered WiFi was not found.

Clicking on the "Ping to Host" link opens a new page where you can enter the IP address or domain of the host you want to ping. If the entered information is correct, a graph displaying ping number and time will be shown. Additionally, details such as host name, minimum, maximum, and average time along with packet loss will be listed.

Clicking on the "Internet Speed Test" link initiates a test of the current network's internet speed. After the test is complete, a link will be provided containing the results. Users can open the link to view details such as download speed, upload speed, and other relevant information about the internet speed of the connected network.
